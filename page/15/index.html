防止与anaconda中的环境冲突设置alias
在~/.bash_profile(如果使用zsh需要在~/.zshrc中)中添加alias python_mac="/usr/local/Cellar/python/3.7.5/bin/python3"
然后通过python_mac安装包
python_mac -m pip install scikit-image
